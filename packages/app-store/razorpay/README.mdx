# Razorpay

Razorpay is a payment gateway service provider based in India that allows businesses to accept, process, and disburse payments with its product suite.

## Installation Steps

1. Visit [Razorpay Dashboard](https://dashboard.razorpay.com) and create an account
2. Once logged in, go to Settings -> API Keys
3. Generate a new API key pair
4. Copy the Key ID and Key Secret
5. In Cal.com, go to the Razorpay App settings and paste your API keys

## Test Mode

For testing purposes, you can use Razorpay's test mode keys. In test mode, you can use the following test card details:

- Card Number: 5267 3181 8797 5449
- Expiry: Any future date
- CVV: Any 3 digits
- OTP: 1234

More test cards are available in the [Razorpay documentation](https://razorpay.com/docs/payments/payments/test-card-details/).
